This process represents a mortgage documentation preparation workflow used to assemble and send supporting documents related to a mortgage package. The process begins with sending the main mortgage document to the recipient.

After the main document is sent, the organization evaluates whether Supplement A must also be included in the package. If Supplement A is required, the process sends Supplement A and then verifies whether Supplement B should also be sent.

If Supplement B is necessary, the organization sends Supplement B before completing the process. If either Supplement A or Supplement B is not required, the process skips the corresponding document delivery activity and directly concludes the workflow.

The process therefore manages conditional document preparation and delivery activities based on the specific requirements of the mortgage documentation package.
